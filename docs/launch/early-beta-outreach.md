# Early Beta Outreach

Hi - we're running an early run-aware public beta for Paperclip Debug MCP.

If you're open to a short evaluation, we'd value practical feedback on three things:

1. setup clarity in your authenticated environment
2. whether the run-aware flow (`list_runs`, `get_run_events`, `trace_handoff`) is useful in real triage
3. whether handoff tracing and incident packets improve escalation quality

We're intentionally keeping scope disciplined: validated for matching authenticated deployment profiles, not a universal compatibility claim. If this sounds relevant, I can share the quick-start and beta docs.
